¡Convierte tu Web en un Espacio Dinámico con Flexbox!

Todos sabemos que una página web se construye a partir de cajas, estas son el esqueleto que da estructura a nuestro diseño. En la creación de esta estructura, tenemos a nuestra disposición varias herramientas. Hoy te mostraré cómo trabajar con las cajas flexibles para construir eficientemente tu diseño. Pero antes de sumergirnos en eso, permíteme mostrarte algunos ejemplos de otras técnicas para que puedas apreciar la gran diferencia entre ellas.

La propiedad position es muy conocida para estructurar nuestras cajas. Con position: relative en la caja padre y position: absolute en las cajas hijo, todas las cajas hijo quedan ubicadas dentro del contenedor padre. Pero, ¿qué sucede cuando reducimos el tamaño de la ventana del navegador? Verás que nuestra estructura cuidadosamente construida se derrumba. Aquí te dejo un ejemplo de código para que puedas experimentar por ti mismo lo que sucede.

¿Cómo declarar la caja padre Flexible?

Para evitar el colapso que acabas de observar, vamos a utilizar el valor flex. Con flex, las cajas se colocan automáticamente una al lado de la otra, de manera sencilla. En este ejemplo, cambiaré un poco las dimensiones en comparación con el ejemplo anterior, pero recuerda, estas son solo sugerencias. A continuación, te dejo el código para que puedas seguir con mayor facilidad.

Para empezar con Flexbox, debemos establecer el contenedor principal, también conocido como caja padre, asignándole display: flex;. De no hacerlo, las cajas hijo se mostrarían en una pila vertical, como es típico en la visualización estándar del navegador. Posteriormente, al aplicar justify-content: center;, conseguimos alinear las cajas hijo horizontalmente al centro del contenedor padre. Es impresionante cómo con solo estas dos propiedades podemos lograr tal estructura.

¿Cómo hacer que las Cajas hijos también sean Flexibles?

Quiero mostrarte otro ejemplo con las cajas flexibles. Esta vez, no solo ajustaremos el contenedor, sino que también permitiremos que el contenido se redimensione a medida que cambiamos el tamaño de la ventana del navegador. Aquí te dejo el código y te lo explico mientras lo revisas.

Como puedes observar en la imagen de arriba, el único cambio en el código CSS con respecto al ejemplo anterior, es que a las cajas hijo no le hemos asignado un valor para el ancho (width). En su lugar, utilizamos la propiedad flex. Esta propiedad es la que permite que las cajas hijo se redimensionen de acuerdo con el espacio disponible. Ahora, ¿qué pasa si quisieras que la caja del medio tenga el doble de ancho que las otras? Ahí es donde flex: 2; entra en juego. Ajustar el valor de flex permite controlar la proporción del espacio que cada caja hijo debe ocupar.

Si sumamos los valores de flex de las tres cajas, obtenemos un total de cuatro. Para calcular el ancho de la caja-1 y caja-3, sería 100% (con respecto a la caja padre) dividido entre 1/4, lo que da como resultado 25%. Para la caja-2, calculamos 100% dividido entre 2/4 (ya que le asignamos dos de las cuatro partes existentes), lo que da como resultado 50%. Repasando lo Aprendido

En este artículo, exploramos las maravillas del Flexbox y su capacidad para estructurar y redimensionar cajas en diseño web. A continuación, he preparado un resumen que destaca los puntos más importantes que discutimos, facilitando una revisión rápida y efectiva de los conceptos clave.

🚀 Resumen Rápido: Desentrañando el Flexbox

  1. La naturaleza de las páginas web: Las cajas son el esqueleto de nuestra página, y el Flexbox ofrece una herramienta versátil para manipularlas.
  2. Propiedad position: Se discutió cómo las cajas hijo quedan ubicadas dentro del contenedor padre y los problemas que surgen al reducir el tamaño de la ventana.
  3. Declaración del contenedor Flex: Aprendimos que con la propiedad `display: flex;` en la caja padre, las cajas hijo se organizan automáticamente de forma horizontal.
  4. Centrar elementos con Flexbox: Utilizamos `justify-content: center;` para centrar las cajas hijo dentro del contenedor padre.
  5. Cajas hijo flexibles: Introdujimos el concepto de cajas hijo redimensionables, que se adaptan según el espacio disponible con la propiedad `flex`.
  6. Controlar proporciones con flex: A través del valor `flex`, podemos controlar el espacio que cada caja hijo ocupa, logrando un diseño más dinámico y adaptable.
  7. Calculando el ancho de las cajas: Vimos cómo el valor `flex` determina la proporción del espacio que cada caja ocupa dentro del contenedor padre.
