¿Cómo ocultar y mostrar contenido con JavaScript?

Cuando llegues al final de este blog, tendrás tu menú de página web completamente listo: responsive y desplegable gracias a JavaScript. ¡Te lo garantizo! Es posible que ya te sientas cómodo con HTML y CSS, pero si estás dando tus primeros pasos en el mundo de JavaScript, no te preocupes. Aquí en AprendeCode, estamos aquí para ayudarte a hacer esos primeros pasos, que a veces pueden parecer desafiantes, mucho más accesibles y fáciles de dar. En realidad, lo que hace que esos primeros pasos sean complicados es el temor a comenzar. Sin embargo, una vez des esos primeros pasos, te darás cuenta de que no es tan intimidante como pensabas. Pero dejemos a un lado esta introducción y centrémonos en lo verdaderamente emocionante: ¿cómo podemos crear un menú desplegable usando HTML, CSS y JavaScript?

A continuación, encontrarás el código HTML y CSS, con algunos estilos predefinidos, que servirá como base. A partir de aquí, nos sumergiremos en la creación de un menú completamente desplegable y adaptable, lo cual es el enfoque principal de esta guía.

¡ Primeros pasos !

Al revisar el código, notarás que hasta ahora no hemos abordado el desplegable en sí, sino que hemos trabajado en los estilos que utilizaremos. Los enlaces actualmente no redirigen a ningún lugar y les he añadido un efecto hover para darle un toque más interesante: ajusté tanto el color como el tamaño con "transform: scale()".

Después de analizar el código y ponerte al tanto de la situación actual, vamos a dar los primeros pasos en el orden adecuado para ir configurando el menú desplegable. El primer paso en la creación de un menú desplegable es ocultar la navegación utilizando Media Queries en el punto que consideremos adecuado. En este caso, he optado por hacerlo en un ancho de pantalla de 1250 píxeles. Ten en cuenta que este valor puede variar de una página a otra, según el tamaño del menú en cuestión. ¡ Ocultando la navegación !

Vamos a empezar a aplicar los estilos en ese punto de quiebre (1250px). Si sigues el mismo orden que yo, podrás observar cómo el menú va cambiando y será mucho más sencillo comprender por qué decidí añadir esos estilos en particular. Comencemos con un "position: absolute" para colocarlo en la ubicación precisa que consideramos conveniente. Mi idea al diseñar este menú es que se despliegue debajo de la barra del menú que estará visible todo el tiempo, dando la impresión de que esta barra se está expandiendo. Para lograrlo, le he añadido un "top: 100px" y tanto "left" como "right" de 30px, ya que toda la página cuenta con un relleno de 30px que marca el inicio y fin del contenido de la página. Luego, empleé "display: block" para que las páginas se posicionen una debajo de la otra, y "text-align: center" para mantener el texto centrado. Hasta aquí hemos establecido la posición y el aspecto deseado, pero aún necesitamos ocultar el contenido por completo. Con este propósito, he incorporado "padding: 0px", "height: 0px" y "overflow: hidden" para evitar que el texto se salga del contenedor. ¡ Creando el ícono del Menú Desplegable !

Hemos logrado ocultar la navegación en pantallas más pequeñas desde ese punto en adelante. Sin embargo, ahora necesitamos incorporar un ícono que permita que la navegación reaparezca cuando el usuario lo seleccione. Para crear este ícono, utilizaremos HTML y CSS para aplicar transiciones. Primero, dirígete a tu archivo HTML y crea un "div" con la clase "icono_menu_desplegable". Dentro de este "div", agrega tres "div" adicionales, cada uno representando una línea. Asignaremos a estos tres "div" una clase común llamada "desplegable_linea" y una clase única para identificar cada uno de ellos: "desplegable_linea_uno", "desplegable_linea_dos" y "desplegable_linea_tres". A continuación, encontrarás el código hasta este punto para que puedas revisarlo. En breve, añadiré los estilos CSS para el ícono, y luego te explicaré los pasos que he tomado.

¡ Estilos del ícono del Menú Desplegable !

En el código, notarás que los estilos para el ícono se han agregado dentro del punto de quiebre de 1250px, asegurándonos de que solo se apliquen desde ese punto en adelante. Ahora, permíteme explicarte un poco sobre los estilos, por qué los he aplicado de esta manera y luego nos adentraremos en el JavaScript. Dentro de la clase "icono_menu_desplegable", lo más relevante es la propiedad "position: relative", que controlará la posición del ícono. Continuamos con un "width" de 49px, un "height" también de 49px, y un "padding" de 12px y 8px.

Luego, encontramos los estilos para las líneas utilizando la clase "desplegable_linea". Comenzamos con "position: absolute", ya que queremos controlar la posición de cada línea de manera individual. A continuación, asignamos un "width" de 34px, "height" de 3px, "border-radius" de 5px y "background-color" blanco. Seguidamente, abrimos llaves individuales para cada clase de línea ("desplegable_linea_uno", "desplegable_linea_dos" y "desplegable_linea_tres") y les asignamos posiciones específicas: "top: 15px" para la primera línea, "top: 23px" para la segunda y "bottom: 15px" para la tercera.

Si has copiado el código en tu editor y ajustas el tamaño de la ventana, podrás ver cómo el ícono del menú desplegable aparece en el punto de quiebre de 1250px y desaparece cuando la ventana se agranda.

¡ Clase desplegar_menu !

Ahora, es necesario que cuando un usuario haga clic en el ícono, el menú se despliegue o, en caso de estar desplegado, vuelva a ocultarse. Este comportamiento se logra mediante JavaScript. Sin embargo, antes de adentrarnos en JavaScript, debemos crear una clase que contendrá los estilos del menú cuando esté completamente desplegado. Para hacerlo, dirigámonos a la hoja de estilos y creemos esa clase, que más tarde añadiremos utilizando JavaScript.

Esta clase contendrá únicamente dos propiedades, ya que los estilos que ya se habían aplicado al contenedor de las páginas son los adecuados. La clase se llama "desplegar_menu". Dentro de ella, solo necesitamos contrarrestar el "padding: 0px" que se estableció previamente con un valor de "30px" y el "height", que originalmente era "0px", ahora se establecerá en "10em". Por último, en la clase "contenedor_paginas", añadí la propiedad "transition", que permitirá que el menú aparezca y desaparezca gradualmente, como ya mencioné anteriormente.

¡ Comienzo del JavaScript !

Una vez definida esta clase, nuestro objetivo es añadir la clase "desplegar_menu" al contenedor de las páginas cuando el usuario haga clic en el ícono del menú. Para lograr esto, utilizaremos JavaScript. Así que, dirijámonos a la sección de JavaScript y empecemos.

Primero, crearemos una variable llamada "iconoMenu" para seleccionar el ícono del menú. Esto se hace con la función "document.querySelector (".icono_menu_desplegable")". Luego, estableceremos una función llamada "desplegarMenu", aunque inicialmente estará vacía: "function desplegarMenu(){}". Posteriormente, fuera de las llaves de esta función, añadiremos un Event Listener a la variable "iconoMenu" que creamos anteriormente. Este Event Listener estará configurado para que cuando se haga clic en el ícono del menú, se active la función "desplegarMenu": "iconoMenu.addEventListener ('click', desplegarMenu)".

Luego, nos adentraremos en la función "desplegarMenu". En su interior, indicaremos lo que deseamos que esta haga. Básicamente, cada vez que se haga clic en el ícono del menú, queremos agregar la clase "desplegar_menu" al contenedor de las páginas que ya contiene la clase "contenedor_paginas". Si la clase ya está presente, deseamos que se elimine para ocultar nuevamente el menú. Para llevar a cabo esto, emplearemos la propiedad "classList" y el método "toggle", que funciona como un interruptor, alternando entre agregar y eliminar la clase de manera continua.

Declararemos una variable llamada "menuDesplegable" utilizando la función "document.querySelector (".contenedor_paginas")". A continuación, añadiremos la propiedad "classList" seguida del método "toggle" a esta variable, de la siguiente manera: "menuDesplegable.classList .toggle("desplegar_menu")".
