Agregar la clase visible al elemento es el siguiente paso. Simplemente utilizamos el método classList.add() en el elemento que queremos que tenga el efecto Fade In. Quedaría de esta manera: elemento.classList.add("visible"). Sin embargo, aquí entra un pequeño matiz importante. Si agregamos esta línea de código tal como está, el efecto Fade In se activaría cada vez que se haga scroll, incluso si el elemento no está a la vista. Esto no es lo que queremos, porque aunque el efecto estaría ocurriendo técnicamente, no sería realmente visible para el usuario.

Para que el efecto sea visible y tenga el impacto deseado, necesitamos que la clase se agregue cuando el elemento esté en una posición en la que sea perceptible para el usuario. Entonces, el primer paso dentro de esta función es calcular la distancia entre el elemento y el borde superior del viewport (área visible) del usuario. Pero no te preocupes, esto es más simple de lo que parece. Te lo explicaré en detalle a continuación.

Utilizando la Propiedad innerHeight para Calcular Distancias

¿Cómo podemos determinar la distancia entre el elemento y el Viewport? ¡Es realmente sencillo! Primero, declaramos una variable llamada "distancia" que recogerá esta medición. Para calcularla, simplemente restamos la altura del Viewport (la ventana visible) a la distancia desde la parte superior de la ventana hasta el elemento. En otras palabras, si imaginas que tu ventana está maximizada al 100%, esto representaría el borde superior de tu monitor o del dispositivo en el que estás leyendo. Pero, ¿cómo podemos obtener la altura de la ventana? Aquí es donde entra en juego la propiedad "innerHeight". Cómo Usar getBoundingClientRect para Optimizar el Efecto Fade In

Espero que estés siguiendo el ritmo hasta este punto. Si algo no está claro o necesitas un poco de orientación, no dudes en dejarme un mensaje en la pestaña contacto. ¡Continuemos! Ahora nos enfocaremos en determinar la distancia entre el elemento al que queremos aplicar el efecto y la parte superior del viewport. Te prometo que esto es tan simple como respirar en el mundo del código. Para lograrlo, tenemos a nuestro aliado: el método getBoundingClientRect(). Con este método, podemos obtener tanto la altura como el ancho del elemento al que lo aplicamos. Por ejemplo, si quisiéramos saber el ancho del elemento contenido en la variable "elemento", podríamos usar elemento.getBoundingClientRect().width. Si ejecutas esto en la consola, obtendrás el ancho del elemento. Lo mismo se aplica para obtener su altura.

Pero aquí está la joya de la corona, al menos para nuestro propósito en este momento. Mediante este método, podemos averiguar la distancia entre los bordes del elemento y el borde del viewport. ¡Impresionante, ¿verdad?! Además, podemos acceder a valores como el margen superior (top), el margen derecho (right), el margen inferior (bottom) y el margen izquierdo (left). Dado que necesitamos la posición del margen superior del elemento, utilizaremos top.

En el código que te proporciono a continuación, estoy pasando la variable "distancia" por la consola para que puedas entender mejor cómo funciona.

Ahora es momento de poner en práctica lo que hemos aprendido. Copia el código que mencioné anteriormente en tu editor de código y abre el inspector del navegador. Verás cómo se aclaran las cosas aún más. En la consola, se mostrará un número que cambiará en tiempo real conforme hagas scroll, sin importar si vas hacia arriba o hacia abajo. Inicialmente, este número será negativo porque la distancia del elemento es mayor que la altura de la ventana. Sin embargo, en el momento en que el borde superior del elemento toque el borde inferior de la ventana, el número se volverá 0. A partir de ahí, comenzará a aumentar en positivo.

Este número será tu guía para decidir en qué punto exacto deseas que el elemento sea visible. Por ejemplo, si el número es 400, significa que, aunque el elemento aún no es visible, ya está a 400 píxeles por encima del borde inferior de la ventana. Esta es una manera eficaz de controlar cuándo quieres que aparezca el elemento a medida que los usuarios realizan scroll. ¡Es como magia!

Cómo Utilizar Condicionales 'if-else' para Controlar el Efecto Fade In

Ahora solo queda un paso para completar este emocionante efecto. Presta atención a la condición que vamos a crear para que el elemento se vuelva visible. Aquí va el último paso para hacer realidad el efecto FADE IN en tu página web. Tan solo necesitas crear una condición que diga lo siguiente: cuando la distancia sea igual o mayor a 600 (en mi ejemplo), agregaremos la clase "visible" al elemento. La lógica es sencilla: cuando la distancia sea mayor o igual a 600 píxeles (o el valor que elijas), el elemento se hará visible aplicando la clase correspondiente. Si deseas que el efecto ocurra en otro punto de la página, simplemente ajusta el valor en la condición. ¡Veamos el código completo para que puedas probarlo por ti mismo!

Después de copiar el código, inicia la depuración y observa cómo cobra vida ante tus ojos. Una vez que la distancia alcanza 600 píxeles (o el valor que hayas elegido), el elemento se mostrará. Sin embargo, es posible que notes que después de que el efecto ocurre una vez, no se repite cuando vuelves a hacer scroll hacia abajo. Pero no te preocupes, esto no es un error. Muchos sitios web diseñan sus efectos para que solo ocurran una vez. Si deseas que el efecto se repita cada vez que haces scroll, puedes ajustar la lógica en la condición. Si quieres que el efecto sea infinitamente repetible, simplemente añade una parte "falsa" a la condición. Es decir, si la distancia no es mayor o igual a 600 píxeles, elimina la clase "visible". ¡Y aquí tienes el código completo para que lo veas en acción!
