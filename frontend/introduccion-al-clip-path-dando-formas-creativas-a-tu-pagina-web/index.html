Introducción al Clip-Path: Dando Formas Creativas a Tu Página Web

Sabemos que la estructura de una página web se compone de cajas. Estas suelen ser rectangulares o cuadradas, pero ¿qué si deseamos darles una forma distinta? ¿Es posible? ¡Por supuesto! Aquí es donde interviene la propiedad Clip-Path. Piensa en ella como una herramienta de recorte que define puntos específicos en la caja, ocultando todo lo que queda fuera de esos puntos. Una vez entendido el concepto, que es bastante intuitivo, es momento de pasar a la práctica. Comenzaremos declarando el contenedor para la imagen y un texto alrededor. A continuación, te proporciono el código para que lo repliques y entiendas con mayor claridad.

Configuración Inicial: Estableciendo la Base

Habiendo dado el primer paso, profundicemos en cada detalle. Los primeros dos contenedores están diseñados para aportar un atractivo visual. El contenedor principal envuelve todo y le añadí propiedades para que resalte. El segundo contenedor es específicamente para el texto e imagen, asegurando que estén centrados y ocupen solo el 90% del espacio principal. Lo crucial aquí es el contenedor de la imagen y el texto. Si replicaste el código, observarás una imagen en su forma estándar. Ahora, emplearemos Clip-Path para darle forma de círculo. Aunque un border-radius podría lograr un efecto similar, usaremos Clip-Path para que te familiarices con ella. Abajo, encontrarás el código e imagen resultante.

Dándole Vida a la Imagen con Clip-Path

Si examinas el código, notarás que simplemente añadimos Clip-Path al contenedor. Aunque parece fácil, es porque es un círculo. Si quisiéramos crear un polígono, la declaración sería más desafiante. Para simplificar esta tarea, existen herramientas como Clippy. Al entrar en el enlace anterior, verás una representación visual como la imagen a continuación.

Refinando el Diseño: Shape-Outside y Shape-Margin

Sin embargo, hay aspectos que podemos perfeccionar. Observa que el texto mantiene la forma original de la imagen. Pero, podemos hacer que adopte la forma del Clip-Path usando la propiedad shape-outside. A continuación, te muestro el código y su resultado. En este caso, usaré un polígono para que veas la utilidad de la herramienta Clippy que mencioné antes.

La solución es añadir border-box al final de shape-outside, recuperando así su forma original. Una vez solucionado, notarás que el texto está muy pegado a la imagen.
