Manipulando el DOM: Creación y Adición de Elementos HTML con JavaScript

Introducción a la Creación de Elementos HTML a través de JavaScript

¿Cómo crear elementos HTML con JavaScript? Para lograr la creación de elementos mediante JavaScript, hay dos métodos fundamentales que es crucial conocer. En primer lugar, tenemos el método createElement(), que, como su nombre sugiere, permite crear un nuevo elemento HTML, como un div, por ejemplo. En segundo lugar, encontramos el método appendChild(), que se encarga de agregar este nuevo elemento, o contenedor, como un hijo dentro del elemento al que se le ha pasado como argumento. Aunque pueda sonar un poco complejo al principio, te aseguro que no lo es en absoluto. Vamos a abordar estos métodos paso a paso para que no te pierdas ningún detalle.

Dentro de esta función, nuestro primer paso es borrar el contenido existente dentro de la caja padre, ya que deseamos que los elementos del array se pinten en su lugar. ¿Cómo lo hacemos? Es realmente sencillo. En primer lugar, creamos una variable para seleccionar la caja padre y poder modificar su contenido. Esto se realiza de la siguiente manera: var cajaPadre = document.querySelector(".caja-padre"). El segundo paso consiste en modificar el contenido de esta caja utilizando la propiedad innerHTML. En nuestro caso, utilizamos comillas dobles vacías para asegurarnos de que el contenido se borre por completo. Es importante recordar que, si deseamos que se muestre un texto literal, lo colocamos entre comillas. Iterando Elementos: El Poder del Método forEach()

Ahora, el siguiente paso consiste en pintar los elementos que se encuentran dentro del array. Dado que estamos tratando con más de un elemento, utilizaremos el método forEach(). ¿Cuál es la función del método forEach()? Este método ejecuta una función para cada uno de los elementos que componen un array o una lista de nodos. La estructura se ve de la siguiente manera: 'letras.forEach(function(elemento){})'. Aquí es importante notar que estamos recogiendo el elemento, lo que significa que le estamos diciendo: Guarda aquí el elemento al que le estás ejecutando la función.

Sin embargo, si intentáramos pasar el elemento directamente a la caja padre, solo se mostraría el último elemento del array. Esto ocurre porque la función se ejecuta de manera muy rápida para cada elemento, y solo se puede visualizar cuando deja de ejecutarse. Ejecuta la función primero para la letra A, la muestra, y luego pasa a B, C y por último D. Puedes realizar una prueba muy sencilla: si eliminas la letra D, solo se mostrará la letra C porque es la última en ejecutarse.

Dinamismo en el DOM: Introducción a createElement()

Aquí, lo que necesitamos hacer es crear un contenedor cada vez que se ejecute la función, y este contenedor será un div. Luego, debemos colocar el elemento que estamos recogiendo cada vez que se ejecuta la función dentro de ese div, y finalmente, añadir ese div recién creado dentro de la caja padre. Comprendo que esto puede resultar un poco confuso al principio, así que vamos a desglosarlo paso a paso. En primer lugar, creamos un div y lo almacenamos en una variable para poder utilizarlo posteriormente. ¿Cómo logramos esto? Simplemente ejecutamos: var cajaHijo = document.createElement(¨div¨).

En segundo lugar, para colocar el elemento dentro de ese contenedor, utilizamos la propiedad innerHTML, de manera similar a como limpiamos el contenido de la caja padre anteriormente. Esto se realiza mediante: cajaHijo.innerHTML = elemento. Por último, en el tercer paso, ponemos ese contenedor, junto con el elemento que le corresponde, dentro de la caja padre. ¿Cómo lo hacemos? Esto es posible gracias al método appendChild(), que se utiliza de la siguiente manera: cajaPadre.appendChild(cajaHijo). A continuación, te dejo el código hasta este punto.

Refinando la Manipulación del DOM usando innerHTML

Si ya has copiado el código, seguramente te habrás dado cuenta de que sí, funciona, pero no exactamente como esperábamos. Todas las letras están dentro de la caja padre, lo cual es correcto, pero esto no se asemeja en nada a lo que teníamos al principio en el HTML. La forma correcta de hacer esto se asemeja más a la estructura HTML original. Si revisamos el HTML, recordarás que teníamos la caja padre, seguida de un div con la clase caja-hijo, y dentro de ese div, la letra (elemento). Eso es exactamente lo que queremos replicar en JavaScript. En lugar de simplemente pasarle a la variable cajaHijo el elemento (o sea, la letra), vamos a crear un div con la clase caja-hijo, luego lo cerramos y, finalmente, dentro de ese div, colocamos el elemento. En primer lugar, debemos agregar comillas alrededor de cada fragmento de string por separado. Esto significa que debemos tomar la etiqueta de apertura del div y colocarla completa entre comillas.
