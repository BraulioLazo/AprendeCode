Incorporando y Personalizando Videos en HTML: Una Guía Completa

Introducción a la Inserción de Videos en HTML

Para incorporar un video en tu página HTML, simplemente necesitas utilizar el elemento de video. Este elemento se compone de una etiqueta de apertura y una de cierre. Dentro de la etiqueta de apertura, se utiliza el atributo SRC para especificar la dirección URL del video que deseas cargar. No obstante, esta acción por sí sola no es suficiente para tener un video listo para reproducir. De hecho, solo hará que el video aparezca en el documento, manteniendo su tamaño original y potencialmente desorganizando la estructura del resto de la página. Es similar a lo que sucede cuando insertas una imagen sin aplicar ningún estilo CSS. A lo largo de este blog, aprenderás no solo cómo insertar un video, sino también cómo personalizarlo y agregar todos los controles de reproducción necesarios.

Como de costumbre, te proporcionamos un código inicial sobre el cual construirás gradualmente en este blog. Lo encontrarás a continuación y lo explicaremos en detalle más adelante. Si eres un amante del código, copia y pega esto en tu editor para seguir el proceso junto con nosotros.

Los estilos necesarios ya se encuentran definidos en el CSS, por lo que no tenemos que realizar ningún cambio adicional en esa sección; todo está listo tal como está. Además, los videos, una vez insertados, serán totalmente adaptables al tamaño de la pantalla, lo que significa que serán completamente responsivos. Usando el Elemento 'source': Adaptabilidad para Varios Formatos de Video

Ahora, me gustaría hacer una pausa para explorar otro método de carga de videos que permite al navegador elegir el formato adecuado. Aunque en la actualidad es poco probable que encuentres un navegador que no sea compatible con el formato MP4, en el pasado esto era una preocupación válida. Para evitar problemas cuando los usuarios ingresaran desde navegadores que no admitieran este formato de video, se creó el elemento source. Este elemento es muy sencillo y consta solo de una etiqueta de apertura. ¿Cómo funciona exactamente este elemento? En lugar de utilizar el atributo SRC dentro de la etiqueta de apertura del elemento video, debemos emplear la etiqueta source entre las etiquetas de apertura y cierre del elemento video. A continuación, te proporciono un pequeño ejemplo para que puedas comprender mejor cómo se implementa.

Para cargar varios formatos de video, necesitamos usar un elemento source diferente para cada formato. Cada uno de estos elementos debe incluir el atributo SRC para indicar la dirección (URL) del video que deseamos cargar. En el ejemplo que te proporciono a continuación, encontrarás dos formatos diferentes del mismo video. El navegador seleccionará automáticamente el formato que sea compatible.

¿Quieres que tu Video Inicie Solo? Conoce el Atributo 'autoplay'

El primer video que he incorporado al documento sería esencialmente la imagen de la web. Por lo tanto, lo lógico sería que este video se reproduzca tan pronto como el usuario entre en la web. Comencemos por ahí. Para lograr que el video se reproduzca automáticamente cuando un usuario accede a la web, tenemos a nuestra disposición el atributo autoplay. Sin embargo, si agregas este atributo a la etiqueta de apertura y guardas los cambios, es posible que notes que no funciona como esperabas.

Ahora, es posible que te estés haciendo una pregunta muy válida: ¿Entonces, para qué sirve el atributo autoplay? Bueno, querido amante del código, resulta que en algunos navegadores, el video solo se reproduce automáticamente si está en silencio. Al menos, en los dos navegadores que utilizo, este es el caso. Para solucionar este problema, simplemente agrega el atributo muted inmediatamente después del atributo autoplay. De esta manera, una vez que el navegador haya cargado el video, se reproducirá automáticamente sin problemas.

Deja tu Video en Bucle Infinito con el Atributo 'loop'

Ya sabemos cómo reproducir el video de forma automática sin problemas. Sin embargo, ahora nos encontramos con el siguiente inconveniente: en algún momento, este video llegará a su final y, lógicamente, se detendrá. Pero dado que este video es la imagen de la web, necesitamos que se reproduzca una y otra vez, incluso cuando llegue a su fin. Afortunadamente, podemos lograr esto gracias al atributo loop. No importa cuántas veces llegue al final, el video se reproducirá una y otra vez en un bucle infinito. A continuación, te dejo el código hasta este punto para que puedas echarle un vistazo y familiarizarte más con él.

Dando el Control al Usuario: Introduciendo el Atributo 'controls'

Ya sabemos cómo reproducir un video de manera automática, lo cual es muy útil para los videos de presentación de una página web. Pero, ¿qué pasa cuando queremos que el usuario decida cuándo reproducirlo? En ese caso, necesitamos agregar controles al video. Esto también es muy sencillo: solo necesitamos agregar el atributo controls al video al que deseamos añadir los controles de reproducción. En mi caso, voy a agregar este atributo a los tres últimos videos.

Cuando añades este atributo al elemento de video, los controles de reproducción aparecerán automáticamente en la parte inferior del video. Estos controles incluyen una barra de reproducción, un botón de reproducción, controles de volumen, la opción de pantalla completa y una pestaña para configurar la velocidad del video. También hay una opción para reproducir el video como una miniatura. Con esta opción, puedes minimizar la ventana del video y seguir viéndolo como una miniatura que puedes mover por la pantalla (ten en cuenta que esto puede variar según el navegador que utilices). A continuación, te dejo el código hasta este punto.

Personalizando la Miniatura de Tu Video con el Atributo 'poster'

Una vez que hemos añadido los controles de reproducción, también podemos cambiar la imagen que se muestra antes de reproducir el video. Esto es similar a las miniaturas que ves en los canales de YouTube. Subes el video y luego seleccionas una imagen que representa el contenido del video. El atributo que utilizamos para esto es similar al atributo SRC. Simplemente agregas el atributo poster, seguido del signo igual y entre comillas, la dirección de la imagen que deseas que se muestre como miniatura para el video. Ejemplo: poster="Medios/Imagen-Uno.jpg".

Optimizando la Carga de Videos con el Atributo 'preload'

Por último, me gustaría mencionarte el atributo preload. Con este atributo, puedes definir si deseas que el video comience a cargarse tan pronto como se abre el navegador o si prefieres que no lo haga. Este atributo acepta tres valores: El primero es none, que se utiliza para minimizar el tráfico web. El segundo valor es metadata, el cual le indica al navegador que descargue información sobre el video, como su duración, dimensiones y otros detalles necesarios para que, cuando el usuario decida reproducirlo, la reproducción sea fluida. El tercer valor es auto, que indica al navegador que descargue el archivo de video tan pronto como sea posible. Este valor es el predeterminado para el atributo "preload".
