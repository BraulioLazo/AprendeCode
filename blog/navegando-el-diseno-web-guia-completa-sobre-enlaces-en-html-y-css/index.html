Navegando el Diseño Web: Guía Completa sobre Enlaces en HTML y CSS

Introducción a los enlaces en la web

¿Enlaces o Links? Realmente, podemos usar ambos términos. Aunque "link" es simplemente la traducción al inglés de "enlace", es una palabra ampliamente adoptada en el mundo hispanohablante. En este artículo, discutiremos cómo declarar enlaces en HTML y cómo darles estilo en CSS para que no parezcan tan insípidos por defecto.

El primer enlace con el que te topas en una página web es, de hecho, el menú de navegación. Estos son enlaces internos que llevan de una página a otra dentro del mismo sitio. También están los enlaces dentro de los blogs; por ejemplo, las imágenes de las tarjetas del blog que llevan hacia la entrada correspondiente. Además, existen enlaces externos que, por ejemplo, podrían llevarte a redes sociales. Pero no todo se detiene ahí: es posible hacer que un usuario, al hacer clic en un enlace, automáticamente pueda enviarnos un correo electrónico o incluso llamar a nuestro número de teléfono sin tener que introducirlo manualmente.

Creando enlaces internos: el menú de navegación

Después de esta breve introducción, te mostraré paso a paso cómo crear estos enlaces. Comencemos con el enlace interno, específicamente el menú de navegación. Imagina que estamos creando tres páginas: Inicio, Blog y Nosotros. Una vez que sepas cuántos enlaces necesitas, es hora de utilizar el elemento "a" dentro de las etiquetas del cuerpo (body). A este elemento le agregamos el atributo href, el cual nos llevará a la página deseada. Aquí te dejo un ejemplo del código y la imagen correspondiente para una mejor comprensión.

Facilitando la comunicación: enlaces de correo y teléfono

Ahora viene una parte bastante interesante, diría yo, ya que estaríamos proporcionando al usuario un medio para contactarnos de manera sumamente sencilla. Usualmente, estos enlaces (correo y teléfono) los encuentras en el pie de página, por lo que eso es lo que usaremos para el ejemplo. Como ya te mencioné anteriormente, sin importar a qué estamos enlazando, la sintaxis es la misma: abrimos la etiqueta de enlace y dentro le agregamos el atributo href. Sin embargo, ahora nos encontramos con una pequeña variación. Las diferencias realmente no son nada complicadas, se trata simplemente de dos palabras clave: una para enlazar al correo y otra al teléfono.

Para enlazar al correo electrónico, utilizamos la palabra clave "mailto", que va seguida de dos puntos y del correo electrónico al que deseamos que nos escriban (observa el ejemplo en el código de abajo). En el caso del teléfono, utilizamos la palabra clave "tel", y seguimos la misma sintaxis que con el correo electrónico. La función de estas palabras clave es invocar al programa o aplicación correspondiente en tu dispositivo para realizar la acción indicada. Ahora, después de entender bien estos ejemplos, es momento de ajustar los estilos predeterminados de los enlaces. Si no has interactuado con ningún enlace aún, podrás observar que su color es azul por defecto, y una vez que haces clic en él, cambia a un tono morado. Además, estos enlaces vienen subrayados. Francamente, ninguno de estos estilos luce particularmente estético. Empecemos eliminando ese subrayado, algo que podemos hacer con la propiedad text-decoration y asignándole el valor none. A continuación, cambiaremos el color predeterminado antes de que el enlace sea visitado utilizando la propiedad color; en este caso, optaré por el negro. Y para aquellos enlaces ya visitados, elegiremos un tono rojo. Esta elección es solo para que puedas notar el cambio; en muchos diseños, especialmente en menús de navegación, es común que el enlace mantenga su color original incluso después de ser visitado. Esto lo logramos gracias a la pseudo-clase :visited. A continuación, te proporciono el código que muestra cómo implementar estos ajustes estilísticos.
